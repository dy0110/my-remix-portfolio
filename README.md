# dy0110のポートフォリオ 

このリポジトリはdy0110のポートフォリオのリポジトリです。

## 構成

- Remix SPAモード

## Development

開発ビルド

```shellscript
npm run dev
```

## Production

本番ビルド

```shellscript
npm run build
```

### Preview

ローカルで本番ビルドの確認をしたいときは [vite preview](https://vitejs.dev/guide/cli#vite-preview) で確認します。

```shellscript
npm run preview
```

> [!IMPORTANT]
>
> `vite preview` 本番用としては使えません.
